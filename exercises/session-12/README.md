# Session 12: Capstone - AI-Enhanced Data Science Rollout

**Duration:** ~30 minutes
**Focus:** Planning an organizational rollout of AI-assisted data science

## Learning Objectives

- Design a team transformation plan for data science workflows
- Create a phased rollout strategy
- Define success metrics specific to analytics teams

## Exercises

### Exercise 1: Current State Assessment (5 min)

Document your team's current state:
- Top 3 pain points in your data science workflow
- Current tools (R, Python, SQL, BI tools)
- Average time from data request to delivered analysis
- Reproducibility track record (can you re-run last month's analysis?)

### Exercise 2: AI Workflow Selection (5 min)

From Sessions 3-11, select the highest-impact workflows:
- Which Claude commands would save the most time?
- What's the biggest quality improvement opportunity?
- Which workflows would be easiest to adopt first?

### Exercise 3: Phased Rollout Plan (10 min)

Using the template in `rollout-template.md`:

**Phase 1 (Weeks 1-2): Pilot**
- 1-2 data scientists try AI-assisted EDA on current project
- Focus on /RunEDA and /RefactorCode
- Measure time savings

**Phase 2 (Weeks 3-6): Team Adoption**
- Full team uses AI for EDA and code review
- Add /ValidateModel and /DocumentFindings
- Pair programming sessions

**Phase 3 (Weeks 7-12): Optimization**
- Custom commands for team-specific workflows
- Integration with team's data infrastructure
- Establish governance policies

### Exercise 4: Present Your Plan (10 min)

Structure:
1. **The Before** (2 min) - Pain points and baselines
2. **The Plan** (5 min) - Tools, phases, governance
3. **The Impact** (2 min) - ROI projection, success metrics
4. **Q&A** (1 min)

## Completion

Congratulations! You've completed the full training curriculum for AI-assisted data science with R. You should now be able to:
- Set up and configure Claude Workflow for R projects
- Use AI assistance for EDA, modeling, statistics, and validation
- Create and validate analysis plans
- Ensure reproducibility and data governance
- Measure and communicate the impact of AI assistance
