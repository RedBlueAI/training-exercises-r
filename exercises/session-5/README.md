# Session 5: Assisted Coding - EDA, Modeling & Statistics

**Duration:** ~20 minutes
**Focus:** Using AI assistance for core data science tasks

This session contains three parallel exercises, each demonstrating a different AI-assisted coding mode. Complete all three to understand the range of AI assistance available for R data science work.

## Exercises

1. **Exercise 1: Refactor Challenge** (~5 min) - Modernize legacy R code
2. **Exercise 2: Implement Feature** (~5 min) - Build EDA visualizations
3. **Exercise 3: Statistical Brainstorm** (~5 min) - Choose the right test

See the individual exercise files for detailed instructions.
