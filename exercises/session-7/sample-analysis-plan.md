# Using the Sample Analysis Plan

The deliberately incomplete analysis plan is at `knowledge/prd/heart-disease-prediction.md`.

## What Makes a Complete Analysis Plan?

A complete data science analysis plan should include:

1. **Research Question** - Clear, testable hypothesis
2. **Data Description** - Source, size, variables, quality assessment
3. **Missing Data Strategy** - How to handle missing values
4. **Feature Engineering** - Planned transformations and derived variables
5. **Model Selection** - Which algorithms and why
6. **Validation Strategy** - CV approach, metrics, baseline comparison
7. **Sample Size Justification** - Power analysis or precedent
8. **Reproducibility** - Seeds, environment, version control
9. **Data Governance** - Ethics, privacy, data use agreements
10. **Timeline** - Milestones with deliverables
11. **Limitations** - Known constraints and assumptions
12. **Deliverables** - Expected outputs and format

The sample plan is missing several of these sections intentionally.
